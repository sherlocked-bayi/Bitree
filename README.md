# Bitree
一个基于MFC的二叉树遍历和展示小程序

- 通过输入先/中/后序遍历序列，画出二叉树
- 能够判断所输入的先/中/后序遍历序列是否能构成二叉树
- 能够通过动画动态展示二叉树的构建过程
- 可点击测试直接看到效果
